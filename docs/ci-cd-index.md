### Task 1: Run Pipeline Manually
1. Go to GitHub Actions
2. Select "CI/CD Pipeline"
3. Click "Run workflow"

**Details:** [Quick Reference](./ci-cd-quick-reference.md#manual-trigger)

### Task 2: Check Pipeline Status
1. Go to GitHub Actions
2. View latest run
3. Check job statuses

**Details:** [Quick Reference](./ci-cd-quick-reference.md#status)

### Task 3: Download Artifacts
1. Go to workflow run
2. Scroll to "Artifacts"
3. Click to download

**Details:** [Full Documentation](./ci-cd-setup.md#artifacts)

2. Review error messages
3. Consult troubleshooting guide
